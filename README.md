# 🔐 Cybersecurity Toolkit

A web-based cybersecurity toolkit built using Flask that demonstrates basic security concepts through interactive tools.

## 🚀 Features

- 🔑 Password Strength Checker  
- 🔐 Caesar Cipher Encryption & Decryption  
- 🖼️ Image Encryption (concept demo)  
- ⌨️ Keylogger Simulation (educational)

## 🛠️ Tech Stack

- Python
- Flask
- HTML/CSS

## 📌 What I Learned

- Basics of cryptography (encryption & decryption)
- Password security mechanisms
- How keylogging works (conceptually)
- Building full-stack apps using Flask
- Structuring modular Python projects
